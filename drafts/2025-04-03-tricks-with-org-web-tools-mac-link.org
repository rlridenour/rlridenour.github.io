I've used Aim√© Bertrand's [[https://gitlab.com/aimebertrand/org-mac-link][Org-Mac-Link]] for some time now. It's incredibly useful for things like saving links to an org document that I use for web bookmarks and adding feeds to my elfeed-org file. I recently stumbled across Adam Porter's [[https://github.com/alphapapa/org-web-tools][Org-Web-Tools ]], which is an extremely useful tool for saving web pages to Org mode files. I've been using a Mac app called SingleFile to save copies of web pages that are useful for research, but since I'm really only concerned with the text, an Org file would be more useful.  I could copy the link in Safari,  use =org-web-tools-read-url-as-org= to create the Org document, and then save it. That's not difficult, but since this is Emacs, there has to be an easier way.

#+begin_src emacs-lisp
(defun rlr/save-web-page-as-org-file ()
(interactive)
(org-mac-link-safari-get-frontmost-url)
(setq rlr-org-link (current-kill 0 t))
(setq rlr-org-link (s-chop-left 2 rlr-org-link))
(setq rlr-org-link (s-chop-right 2 rlr-org-link))
(setq rlr-org-link (s-split "\\]\\[" rlr-org-link))
(setq rlr-org-url (pop rlr-org-link))
(setq rlr-org-title (pop rlr-org-link))
(setq rlr-org-title (s-replace-all '(("." . "") (":" . "") ("/" . "")) rlr-org-title))
(setq rlr-org-filename (s-dashed-words rlr-org-title))
(org-web-tools-read-url-as-org rlr-org-url)
(write-file (concat "~/icloud/web-saves/" rlr-org-title ".org")))
#+end_src

(defun rlr/open-safari-page-in-eww ()
(interactive)
(org-mac-link-safari-get-frontmost-url)
(setq rlr-org-link (current-kill 0 t))
(setq rlr-org-link (s-chop-left 2 rlr-org-link))
(setq rlr-org-link (s-chop-right 2 rlr-org-link))
(setq rlr-org-link (s-split "\\]\\[" rlr-org-link))
(setq rlr-org-url (pop rlr-org-link))
(eww rlr-org-url))
#+end_src
